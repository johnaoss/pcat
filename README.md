# pcat

pcat is a pretty `cat` util, applying syntax highlighting to outputted code if possible.

# Installation

Currently requires a 256-bit colour terminal (terminal256). 

todo: provide binary release

# Usage

```
pcat filename.html

pcat < filename.html
```
